    • Version 4.06

    Here's the announcement.

    The documentation has been overhauled. The installation notes are now part of the user guide, and the installation guide has been replaced by a building guide. Both have extensive notes on working under Windows.

    All the libraries have been moved into hslibs, which is intended to become a comprehensive set of libraries that will work with any Haskell 98 compiler. Some new libraries have been added, including the Edison data structures library.
